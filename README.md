# A Minimal Model for Turn Taking in Group Discussions

This repository is a copy of the computational capsules originally hosted 
on the Code Ocean platform. The code is used to reproduce the figures 
presented in the paper:

**“A Minimal Model for Turn Taking in Group Discussions”**

## Software Requirements

* The Python notebooks were developed for **Python 3.9**.
* The MATLAB scripts were developed for **MATLAB R2023 or later**.

## Figure Reproduction

The scripts and notebooks used to generate each figure are listed below:

* **Figure 1:**
  `jupyter_notebook_code/code/Visualize_Speaking_Time.ipynb`

* **Figure 2:**
  `jupyter_notebook_code/code/Reciprocal_pattern_submission.ipynb`

* **Figure 3:**
  `jupyter_notebook_code/code/Demo_submission.ipynb`

* **Figure 4:**
  `matlab_code/code/fitmodel.m`

* **Figures 5 and 6:**
  `jupyter_notebook_code/code/Model_evaluation.ipynb`

## Code Ocean Executables

The executable scripts included in the original capsules are configured 
specifically for the Code Ocean environment and may only work when run on 
the Code Ocean platform.

When running the code locally, execute the corresponding Jupyter notebooks 
or MATLAB scripts directly instead.
